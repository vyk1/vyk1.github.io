## Course Monitoring and Evaluation System aiming administrative and educational support for institutions within the e-Tec Brasil Network

The Course Monitoring and Evaluation System (SAAS) is a web application developed in Java, using the Spring Framework, Hibernate, and MySQL. The system was developed to be a tool for administrative and educational support for institutions within the e-Tec Brasil Network.
